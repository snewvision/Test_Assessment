Locators:-
In Playwright, locators are the core mechanism to find and interact with elements on a web page (buttons, inputs, links, text, etc.).
Playwright’s locator system is smart, auto‑waiting, and reliable compared to old-style selectors.

These are the recommended built-in locators.
page.getByRole() to locate by explicit and implicit accessibility attributes.
page.getByText() to locate by text content.
page.getByLabel() to locate a form control by associated label's text.
page.getByPlaceholder() to locate an input by placeholder.
page.getByAltText() to locate an element, usually image, by its text alternative.
page.getByTitle() to locate an element by its title attribute.


await page.getByLabel('User Name').fill('John');

await page.getByLabel('Password').fill('secret-password');

await page.getByRole('button', { name: 'Sign in' }).click();

await expect(page.getByText('Welcome, John!')).toBeVisible();

Assertion :- Playwright uses the expect API for assertions.
In Playwright, an assertion is a way to verify that something on the page behaves or appears as expected. If the condition is not met, the test fails.

Basic Idea

An assertion answers questions like:

“Is this element visible?”
“Does this text match?”

  await page.goto('https://example.com');

  await expect(page).toHaveTitle('Example Domain');
